📄 향수 추천 서비스 기획서 (WeatherScent 가칭)
1. 서비스 개요
WeatherScent는 사용자의 현재 위치와 날씨 정보를 기반으로 OpenAI를 활용해 어울리는 향수를 추천하고, 추천된 향수를 실제로 구매할 수 있는 오프라인 매장 또는 온라인 링크를 제공하는 향수 큐레이션 플랫폼입니다. 사용자의 취향, 기분, 날씨, 감정 등 다양한 요소를 조합해 정교한 맞춤형 향수 추천 경험을 제공합니다.

2. 핵심 기능
브라우저의 위치 권한을 통해 사용자의 실시간 좌표를 수집하고, OpenWeather API로 현재 날씨를 조회합니다.

조회된 날씨 정보(기온, 습도, 하늘 상태 등)는 LangChain과 OpenAI에 전달되어 향수 스타일 및 제품명 추천에 활용됩니다.

추천된 향수명은 FragranceFinder API를 통해 브랜드, 노트, 이미지, 유사 향수 등의 정보를 확보합니다.

Kakao Local API를 통해 사용자의 위치에서 가장 가까운 향수 매장을 탐색하여 매장 이름, 주소, 거리, 운영시간을 안내하고 지도에 시각화합니다.

향수 추천과 함께, 해당 날씨나 상황에 어울리는 감성적인 문장을 AI가 자동 생성해 사용자의 몰입감을 높입니다.

사용자는 간단한 취향 테스트(계열 선호, 무게감, 향수 사용 목적 등)를 통해 자신의 향수 취향을 분석할 수 있으며, 이 결과는 추천 알고리즘에 반영됩니다.

향수 추천은 날씨나 취향뿐 아니라 감정 기반 분기로 확장되며, 예를 들어 "기분 전환", "로맨틱한 날", "활기찬 하루", "데일리용" 등의 목적에 맞는 추천이 가능합니다.

사용자는 향수를 찜하거나 보유 목록에 추가할 수 있으며, 향수 상세 페이지에서 향조, 브랜드, 이미지 등을 확인할 수 있습니다.

UUID 기반 추천 결과 링크를 생성해 친구에게 공유할 수 있고, 향수에 대한 리뷰와 별점을 남길 수 있는 기능도 구현 가능합니다.

별도로 제공되는 챗봇 인터페이스를 통해 사용자는 “오늘처럼 흐린 날에 어울리는 향수 추천해줘”와 같은 질문을 입력하고 대화형으로 향수 추천을 받을 수 있습니다.

3. 고급 기능 (구현 가능)
사용자의 성별, 나이, 선호 계열, 목적 등을 바탕으로 OpenAI 프롬프트를 개인화하여 추천 결과의 정확도를 높일 수 있습니다.

이전 추천 이력, 찜한 향수, 자주 클릭한 계열 등을 분석하여 점차 고도화된 개인화 추천이 가능합니다.

향수 추천 결과 외에도 "오늘 같은 날씨엔 이런 감성", "촉촉한 공기 속에 어울리는 편안한 무드" 등 감정 기반 문장을 함께 제공하여 정서적 몰입을 유도합니다.

향수 추천을 감정 기반 분기로 확장하여 “기분 전환이 필요할 때”, “특별한 날”, “일상에서 가볍게” 등 사용자의 목적별 향수 선택을 지원합니다.

취향 테스트는 간단한 선택지 기반 퀴즈 형식으로 제공되며, 결과에 따라 선호 계열, 사용 목적, 지속력, 무게감 등의 기준을 프롬프트에 반영합니다.

추천된 향수의 상세 페이지에는 브랜드, 탑/미들/베이스 노트, 이미지, 연관 향수 등이 포함되며, FragranceFinder API로부터 실시간으로 받아올 수 있습니다.

유사 향수 추천은 향조 유사도 기반으로 구현되며, LangChain의 벡터 검색 또는 OpenAI 프롬프트 활용을 통해 제공합니다.

향수 찜, 보유 목록 저장, 별점 평가, 사용자 리뷰 기능 등을 통해 커뮤니티적인 기능도 구축할 수 있습니다.

사용자의 클릭 로그, 찜 수, 공유 횟수 등을 바탕으로 인기 향수 순위를 집계하고, 지역 기반 인기 향수도 추출할 수 있습니다.

4. 사용 기술
프론트엔드는 Next.js와 TailwindCSS 기반으로 구축되며, 지도 기능은 Kakao Map SDK를 통해 구현됩니다.

백엔드는 Node.js (Express) 기반이며, LangChain으로 프롬프트 템플릿과 Tool Calling을 처리합니다.

날씨 정보는 OpenWeather API를, 매장 탐색은 Kakao Local API를, 향수 정보는 FragranceFinder API를 활용합니다.

사용자, 추천 이력, 찜 목록 등은 PostgreSQL 또는 Supabase를 사용해 DB에 저장됩니다.

패키지 매니저는 pnpm을 사용합니다.

5. 사용자 경험 흐름 예시
사용자가 웹사이트 접속 후 위치 권한을 허용하면 현재 위치와 날씨가 자동으로 수집됩니다.

날씨와 사용자 정보(성별, 취향, 감정 목적 등)를 기반으로 LangChain이 OpenAI에게 프롬프트를 전달합니다.

AI가 향수 계열, 제품명, 추천 이유, 감성 문구를 함께 생성합니다.

FragranceFinder API로 향수 정보가 확보되고, Kakao API로 가까운 오프라인 매장이 검색됩니다.

사용자는 추천 결과를 카드 형태로 확인하고, 찜하거나 공유하거나 향수 상세 페이지로 이동할 수 있습니다.

별도의 챗봇 메뉴에서 "데이트에 어울리는 향수 추천해줘", "겨울에 데일리로 뿌릴 향수 있어?" 등의 대화를 통해 맞춤 추천을 받을 수 있습니다.

6. 확장 방향
감정 기반 큐레이션을 강화해 계절+기분+상황을 모두 반영하는 멀티 레이어 추천 시스템 구축

향수 사용 목적별 추천(출근용, 휴일용, 여행용 등)을 옵션화

향후 제휴 API 또는 크롤링으로 가격 비교, 재고 확인 기능 추가

시향 가능 매장 표시 및 예약 기능 등 오프라인 경험 강화

사용자 테스트 결과 기반 향수 프로필 자동 생성
